# LocalGem 💎

**LocalGem** — это легкое и приватное мобильное приложение для запуска больших языковых моделей (LLM) прямо на вашем устройстве. Проект построен на базе движка LiteRT.

## ✨ Ключевые особенности

*   **100% приватность**: Все вычисления происходят на вашем устройстве. Ваши данные никогда не покидают телефон.
*   **Управление чатами**: Полноценная история переписок, хранящаяся в локальной базе данных SQLite.
*   **Мультимодальность**: Поддержка мультимодальных моделей и режима мышления.
*   **Детальная статистика**: Отслеживание скорости генерации (TPS), времени до первого токена и т.д

## 🛠 Технологический стек

*   **Frontend**: React Native
*   **State Management**: Zustand
*   **Storage**: SQLite (`react-native-sqlite-storage`)
*   **Backend**: LiteRT LM

## 🚀 Сборка и запуск

1.  Установите зависимости: `npm install`
2.  Запустите Metro Bundler: `npm start`
3.  Запустите приложение на Android: `npm run android` или соберите apk в Android Studio
